Project Overview

Multimodal Machine Learning for Startup Hiring Quality

The Problem

Predicting the desirability of startups across three classes: Emerging, Growing, and High Desirability. Initially plagued by target leakage, we redefined the problem as a strict 3-class classification task using only raw business fundamentals and semantic role texts.

The Solution

A novel 3-phase hybrid architecture closing the loop between unsupervised probability and deep learning. By using a GMM's uncertainty to dynamically gate a DistilBERT text stream and a Tabular MLP, we achieve true multimodal synergy.

System Architecture

Phase 3: Entropy-Gated Synergistic Hybrid

The Innovation: The GMM's Shannon Entropy acts as a dynamic attention mechanism, deciding per-sample whether to trust semantic text features (DistilBERT) or structured tabular features (MLP).

Diagnostic Analysis

Solving the "Growing" Class Boundary

Targeting Uncertainty: Model errors are heavily concentrated in high-entropy (uncertain) samples, mostly around the ambiguous "Growing" class. Phase 3 feeds this exact uncertainty into the attention gate to adapt the fusion strategy dynamically.

Interpretability

Transparent Predictions: Culture, Salary, and Work-Life Balance ratings are the dominant predictors, proving the model learned genuine domain fundamentals rather than proxy artifacts.
